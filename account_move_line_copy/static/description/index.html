Duplicate Invoice Line

This module add a button at the end of each invoice line that allows users to quickly duplicate the line.

The code is well-documented, making it easy to customize to your specific needs.
The module is available in both English and French.


Duplicate an invoice line by clicking on the button

The duplicated line appears just under the original line.

This feature is useful for splitting the invoiced quantity of a product at two different rates. The duplicated invoice line retains the link to the sale order line that generated the original invoice line. As a result, the invoiced quantity on the sale order line will be the sum of both the original line and the duplicated line.
